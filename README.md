#  Hangman Game (Console)

A simple console-based Hangman game designed for beginners to practice programming fundamentals such as loops, conditions, arrays/strings, and user input.
##  Features

* Random word selection
* Letter-by-letter guessing
* Limited attempts
* Win or lose conditions
* Simple and readable code

##  How to Run

### Java

bash
javac Hangman.java
java Hangman
##  How to Play

* A word is hidden with underscores
* Guess one letter at a time
* Correct guesses reveal letters
* Wrong guesses reduce attempts
* Guess the word before attempts end to win
